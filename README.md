# CainRubberBullets
Testing plugin for Unturned. Used to test different effects with different ammo types.

# How it works
  Config contains toggles for each bullet type. Also contains list of ids for the ammo items that need to be in the shooter's gun for these effects to work.
  ```Rubber Bullets``` - Breaks character's legs<br>
  ```Tranq Bullets``` - Drops character's water and food, raises virus level and causes hallucinations<br>
  ```Training Bullets``` - Bullets do 0 damage but tell the shooter how far away they are and where they hit them on their body<br>
  ```Zombie Bullets``` - Spawns a zombie on the hit location (Warning, I don't remember if I fixed this but it used to spawn an invisible mega zombie that you couldn't kill. Use at own risk)<br>
  ```Tracking Bullets``` - Places a waypoint on shooter's map of player location. Location will update over time for configurable amount of time.
